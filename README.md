## Descripción

Este proyecto es una aplicación de videojuegos, que tiene como objetivo consumir datos de la API externa [Rawg](https://rawg.io/apidocs) para mostrar los juegos en forma de catálogo.

En esta aplicación, el usuario podrá elegir el videojuego de su preferencia buscando en el catálogo a disposición o utilizando filtros, como género, rating y orden alfabético. Para facilitar la búsqueda en una lista de al menos 100 videojuegos, la aplicación cuenta con un paginado al final de la página principal.
